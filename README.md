##### Variables
`a = 5` Creates a variable named "a" with value "5"


##### Operations
* `+` : `10 + 5`  is equal to  `15`
* `-` : `10 - 5`  is equal to  `5`
* `*` : `10 * 5`  is equal to  `50`
* `/` : `10 / 5`  is equal to  `2`
